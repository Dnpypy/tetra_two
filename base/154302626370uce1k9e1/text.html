Копирование в nano содержимого файла

устанавливаем программу xclip

Ну и далее скопировать содержимое файла в общий (главный) буфер обмена можно так:

cat ./ваш_файл|xclip -i

-- такая шутка скопирует в главный буфер обмена иксов. Вставить можно будет кликнув по т.н. средней клавиши мыши (на практике это обычно колёсико).

Для Ctrl+V же копировать надо так:


cat ./ваш_файл|xclip -i -selection clipboard
