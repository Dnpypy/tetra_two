для 3.3.2 делаем аналогичные операции:

$ wget http://www.python.org/ftp/python/3.3.2/Python-3.3.2.tar.xz ~/python/src/

$ tar -xf ~/python/src/Python-3.3.2.tar.xz && cd ./Python-3.3.2

$ ./configure --prefix=$HOME/python/3.3.2/

$ make && make install
